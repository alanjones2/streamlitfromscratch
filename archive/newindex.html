Streamlit is a framework for creating Data Science apps in Python. Streamlit from Scratch is an evolving onlne resource that will teach you how to get started with Streamlit through to building fully functional data dashboard apps using Streamlit's layout and data presentation features.
The articles and hints on this site will help get you started and will build into a significant resource. (Links to articles that are hosted on Medium are free to read 'Friend Links' that bypass the Medium paywall. )

Streamlit from Scratch an online course

Streamlit from Scratch will guide you from the very start to a point where you are comfortable with creating your own sophisticated data science apps. It will cover:
  • Finding the tools you need and how to use them to create your first interactive web app
  • Adding media to your apps helps to communicate your ideas
  • Presenting data: from text and tables to sophisticated charts. From Pandas dataframes to Matplotlib, Plotly, Altair and other charts. Streamlit provides a comprehensive set of tools for presenting your data
  • build effective and attractive data science and dashboard applications using Streamlit's simple but effective components for user interaction and layout
