Streamlit from Scratch building data science apps with Python

Streamlit is a framework for creating Data Science apps in Python.

Streamlit from Scratch will teach you how to get started with Streamlit in 5 steps:

  • First, we show you the tools that you will need
  • We start coding with a simple text-based web app
  • Then we add media - images, video, and audio
  • Next we present data - charts and maps using Matplotlib, Plotly and other charting tools
  • Finally, we create fully interactive data dashboard apps using Streamlit's simple but powerful layout capabilities
