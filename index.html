Streamlit from Scratch building data science apps with Python

Streamlit is a framework for creating Data Science apps in Python.
Streamlit from Scratch is an online resource that will teach you how to get started with Streamlit
from you first simple text-based web app, then adding media, presenting data, and finally building fully functional data dashboard apps
using Streamlit's layout and data presentation features.
